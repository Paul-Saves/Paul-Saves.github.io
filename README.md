# Academic Portfolio - Dr. Paul Saves. [paul-saves.github.io](https://paul-saves.github.io/).

Welcome to the source repository for my professional academic portfolio,  hosted at [paul-saves.github.io](https://paul-saves.github.io/).

This site serves as a comprehensive hub for my research in Frugal AI, Surrogate Modeling, and Multi-disciplinary Design Optimization (MDO).

## 🛠️ Technical Structure

- `_data/navigation.yml`: Centralized control for the site's header navigation.
- `_layouts/`: Custom HTML templates (`default`, `page`, `post`) providing a consistent academic shell.
- `css/main.css`: Professional typography and layout styling based on IEEE-compliant aesthetics.
- `index.md`: Professional biography and career milestones.
- `research.md`: Detailed research framework, strategic projects, and supervision history.
- `publications.md`: Complete bibliography with direct DOI and ArXiv access.
- `teaching.md`: Pedagogical statement and core curriculum summary.
